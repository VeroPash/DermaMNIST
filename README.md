# DermaMNIST
This project implements a deep learning model to classify pigmented skin lesions into 7 diagnostic categories using the DermaMNIST dataset.
## Highlights
- **Model Architecture:** Fully Connected Neural Network (FCNN) optimized with ReLU activation and Adam optimizer.
- **Data Engineering:** Normalization and one-hot encoding for preprocessing; class imbalance addressed using balanced weights.
- **Performance:** Achieved ~72% validation accuracy on 28x28 medical images.
- **Tools:** TensorFlow, Keras, NumPy, Scikit-learn, Matplotlib.
## Dataset
The project uses the **DermaMNIST** dataset, a subset of the MedMNIST collection consisting of 10,015 dermatoscopic images.
